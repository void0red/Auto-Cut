# Auto Cut

预览视频，按照音乐自动分段。

## Usage
```python
# pip install -r requirements.txt
# 待处理的文件放在video文件夹下，剪切的分段放在cut_videos文件夹下
# 其他配置请参考config.py
python app.py
# 1. 访问 http://localhost:5000
# 2. 选择视频，点击自动分段
# 3. 点击列表跳转预览视频，开始时间，结束时间可编辑，按回车键确认
# 4. 点击开始剪辑视频，完成分片
```
